## use cases that are not being solved with CEC

1. Cleaning up, isolating, translating CEC - "fixing cec"
    * This could mean blocking specific devices or calls.  CEC has protocals built off it.  So CEC as a standard can be messy.
2. [Adding CEC support.](https://www.hackster.io/guorongkoh/pico-cec-hdmi-cec-to-usb-keyboard-adapter-6d18f9)
    * not all devices have CEC, so adding a device in your hdmi cable path could fix that.  
3. [Controlling CEC devices using HASS or similar that arent "smart" / dont have an API or intigration](https://www.domadoo.fr/en/smart-wifi-module/7782-smlight-hdmi-cec-controllerdevelopment-board-for-esphome.html)
    * Some devices might not be smart but still support CEC, so thatd allow you to add it to the smart home ecosystem you have.
